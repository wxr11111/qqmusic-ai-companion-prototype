# QQ音乐 AI 专属陪伴交互原型

本仓库用于归档 QQ 音乐「AI 专属陪伴」功能的交互原型文件。原型以静态 HTML 方式呈现，聚焦播放器主场景下的 AI 陪伴体验设计，用于承载方案评审、需求对齐与产品表达阶段的页面演示。

## 项目背景

「AI 专属陪伴」围绕 QQ 音乐播放器主链路展开，目标是在不打断用户听歌节奏的前提下，引入可感知、可互动、可控的 AI 陪伴能力。原型重点验证以下几个方向：

- AI 能力如何自然进入首页与播放器链路
- 用户如何完成陪伴形象选择与角色确认
- 播放页中如何实现低打扰的轻陪伴体验
- 当前歌曲语境下如何承接深度对话与推荐操作
- 情绪场景下如何以更温和的方式提供关怀反馈

## 原型范围

当前原型共包含 5 个页面，覆盖一条完整的核心体验路径：

1. 01 首页入口页:
   AI 陪伴能力以轻量入口形式进入 QQ 音乐首页内容流。
2. 02 形象选择页:
   用户选择 AI 陪伴形象，建立角色偏好与陪伴感知。
3. 03 播放页默认态:
   AI 在播放器中以轻陪伴状态常驻，提供低打扰提示与控制入口。
4. 04 全屏 AI 面板:
   承接当前歌曲场景下的聊天、追问、推荐与操作闭环。
5. 05 情绪关怀页:
   在特定情绪信号或明确诉求下展示情绪承接能力。
